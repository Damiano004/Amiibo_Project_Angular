# AmiiboApp

AmiiboApp è un'applicazione web sviluppata in Angular (19.2.10) che mostra tutta la lista di amiibo esistenti, la loro provenienza e il loro utilizzo in gioco, e infine rende possibile agli utenti di commentare queste statuette.

## Funzionalità principali

- **Ricerca degli Amiibo**: Puoi cercare il nome degli amiibo specifici che vuoi visualizzre.
- **Filtraggio per Gioco**: Filtra gli amiibo per il loro gioco di provenienza.
- **Commenta**: Dì la tua sui tuoi amiibo preferiti o quelli più utili.

## Prerequisiti

- [Node.js](https://nodejs.org) versione 22.14.0
- [Angular CLI](https://angular.io/cli) versione 19.2.12
- [PrimeNG](https://primeng.org/) versione 19.1.2
- [PrimeNG icons](https://primeng.org/icons) versione 7.0.0

## Avvio progetto

```bash
npm run start
```
